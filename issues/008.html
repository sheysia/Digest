论文解读#04
当 AI 帮你查数据的时候，谁在帮攻击者查你的数据？
📌 2026.06.07 · 来源：arXiv 2606.08661 · 南洋理工/香港理工/清华
★ 亮点说人话

LLM 驱动的 Data Agent（Databricks Genie, Snowflake Cortex 等）正在企业中广泛采用。这篇论文揭示了一个全新攻击面：单条查询无害，组合起来就泄密。三层漏洞框架（解释/执行/策略），14 种攻击技术，6 个系统全部存在实质漏洞。

💡 So What

Agent 安全不只是"别让它说有害内容"。当 Agent 接入真实数据源和执行环境，分析过程本身成为攻击面。企业的数据库权限控制还停留在 Agent 出现之前的范式，这个缺口正在被量化。

Paper Review#04
When AI Queries Your Data, Who Is Querying YOUR Data?
📌 2026.06.07 · Source: arXiv 2606.08661 · NTU / PolyU / Tsinghua
★ Plain-English Highlight

LLM-driven Data Agents (Databricks Genie, Snowflake Cortex, etc.) are widely deployed. This paper reveals a new attack surface: individually harmless queries that leak sensitive data when combined. Three vulnerability layers, 14 attack techniques, all six tested systems have real gaps.

💡 So What

Agent security is not just about harmful content. When agents connect to real data and execution environments, the analysis process itself becomes the attack surface. Enterprise database controls still reflect pre-agent design paradigms.

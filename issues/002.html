工具上新#01
🔥 ChatGPT Images 2.0 — 昨天刚出，AI 作图进入"文艺复兴"
📌 openai.com · Apr 21, 2026
✹ 亮点说人话

OpenAI 昨天（4/21）发布了 ChatGPT Images 2.0，底层模型叫 gpt-image-2。12 小时内登顶 Image Arena 所有类别第一名，领先第二名 242 分——这是该排行榜有史以来最大差距。核心升级：文字渲染准确率从 ~90% 跳到 99%（终于不会把 "WELCOME" 写成 "WELCOOMM" 了）、支持 4K 分辨率、内建推理能力（会"想"再画）、多轮编辑保持一致性。Sam Altman 说"这是从 GPT-3 直接跳到 GPT-5 的感觉。"

💡 So What

对你意味着：做海报、信息图、产品 mockup、社交媒体配图终于可以靠 AI 了——以前文字一乱就没法用，现在文字准了。免费用户有 Instant 模式，付费用户有 Thinking 模式（质量更高但稍慢）。Nano Banana Pro 之后又多了一个强选择。DALL-E 2/3 将在 5 月 12 日正式退役。

所有人需要做配图的人尤其受益

工具上新#03
Nano Banana Pro — 用 Gemini 画科研级流程图和信息图
📌 blog.google · Gemini 3 Pro Image
✹ 亮点说人话

Google 的 Gemini 图像模型 Nano Banana Pro 能从自然语言生成流程图、架构图、信息图——文字清晰、逻辑自动排版。生成后还能对话式修改（"把箭头改成红色""加一个分支"）。免费通过 Gemini app 使用。英文文字渲染最好，中文可能乱码。

💡 So What

以前做一张流程图要开 Visio/Lucidchart 拖半小时。现在打字就出。在 DMV 工作场景下特别好用——report 里的 methodology diagram、项目架构图、对比信息图，英文标注正好是优势。配合刚出的 ChatGPT Images 2.0，你可以两个都试试看谁出的图更好。

👩‍💻 适合谁

所有人需要在 report/PPT 里配图的人

场景拆解#04
AI 写论文/长文 — 从文献到定稿的完整工作流
📌 By Syneira · 深度指南 · Tutorial 003
✹ 亮点说人话

两条路线：路线 A（零代码） Perplexity 找文献 → Elicit 做证据表 → Claude 写大纲和正文（用中英混搭 prompt）→ Writefull 学术润色 → thesify 当你的 AI 审稿人 → Zotero 管引用。路线 B（进阶） Claude Code 直接生成 LaTeX → Overleaf 协作。11 个工具对比表 + 底线原则（哪些绝对不能让 AI 做）。

💡 So What

核心原则：AI 帮你表达和组织思路，不帮你生成论点。你的分析是你的。这份指南不推荐一个"万能工具"，而是让每个工具做最擅长的事——你是导演，AI 是剧组。

场景拆解#05
AI 做 PPT + 高级感 Report + 配图 — 三合一工作流
📌 By Syneira · 深度指南 · Tutorial 001
✹ 亮点说人话

PPT 三条路线：Gamma（60 秒出稿，免费）→ Canva AI（已有习惯不换工具）→ Claude Design / Copilot in PPT（正式对外）。Report 三条路径：Claude Artifacts（你正在看的这份 digest 就是这么做的）→ Notion AI → Gamma Document。配图：Nano Banana Pro / Claude Mermaid / Canva。附三套推荐组合：$0 版 / $38/月版 / 企业版。

💡 So What

核心原则：你给骨架（大纲/要点），AI 填肉（排版/视觉/文案润色）。不要让 AI 从零生成——先想清楚"我要说什么、给谁看"，再让 AI 帮你做得好看、做得快。

用法洞察

用法洞察#07
万能 Prompt 模板 — 一个框架通吃论文/PPT/Report/求职
📌 By Syneira · 实操总结
✹ 亮点说人话

不管你让 AI 做什么，都可以用这个框架：角色（你是一个专业的 [行业] [角色]）→ 上下文（给 [受众] 看的 [文档类型]，目的是 [什么]）→ 任务（基于以下大纲生成 [具体产出]）→ 格式（排版/配色/语言偏好）→ 约束（篇幅/风格/禁止事项）。

💡 So What

大多数人用 AI 效果不好，不是模型的问题，是 prompt 太模糊。同样的模型，好的 prompt 和差的 prompt 产出差距可以是 10 倍。这个框架配合 Issue #001 的中英混搭技巧和 TextGrad 自我优化，就是一套完整的 prompt 方法论。

👩‍💻 适合谁

所有人今天就能用

趋势信号#08
Hermes Agent — AI 终于能"记住怎么干活"了
📌 小红书 @kusengo · GitHub: NousResearch/hermes-agent
✹ 亮点说人话

以前用 AI 就像每次请一个新实习生——你得从头教。Hermes Agent 有内建学习闭环：执行任务 → 自动复盘 → 沉淀成 Skill → 下次直接调用。三层记忆（会话/持久/Skill），GitHub 两个月冲到 10 万 Star。但才 v0.x，不适合生产环境。

💡 So What

即使你不装 Hermes，这个趋势值得关注——AI 正在从"一次性对话"变成"长期合作伙伴"。Claude memory、ChatGPT custom instructions、Codex memory、这周的 Hermes，都是同一方向。未来选 AI 工具，"它能不能记住我、越用越懂我"会成为核心竞争维度。

工具更新#01
Claude Opus 4.8：不只是更聪明，是更诚实
📌 2026.05.28 · 来源：Anthropic 官方博客
★ 亮点说人话

Anthropic 发布 Opus 4.8，定价不变但有三个实质变化：诚实度提高 4 倍（主动标记代码问题而不是礼貌性略过）、agentic coding 能力提升、Claude Code 新增 ultracode 模式和动态工作流。

💡 So What

模型竞争正在从"谁更聪明"转向"谁更可靠"。当能力已经足够完成大多数任务时，诚实度和一致性才是决定用户留存的关键。Opus 4.6 / 4.7 不受影响，继续可用。

工具上新#02
Hermes Agent 官方桌面版来了：开源 agent 从终端走向所有人
📌 2026.06.02 · 来源：Nous Research / Decrypt / MarkTechPost
★ 亮点说人话

上期介绍过 Hermes Agent 的核心理念（持久记忆 + 技能自学），这周它正式发布了官方桌面版（v0.15.2，Public Preview）。macOS / Windows / Linux 全覆盖，一键安装，和 CLI 共享同一个 agent 核心。4 个月 180K GitHub stars，是 2026 年增速最快的开源 agent。

💡 So What

桌面版的意义不只是"多了个 GUI"。它把一个原本只有技术用户能玩的 agent 开放给了所有人。和 Claude Desktop / ChatGPT Desktop 的区别是：Hermes 是"住在你机器上、越用越懂你"的本地 agent，不是一个聊天窗口。

👩‍💻 适合谁

想要长期记忆 AI 的人本地优先 / 隐私关注者多平台工作流

Tool Update#02
Hermes Agent Ships Official Desktop App: Open-Source Agent Leaves the Terminal
📌 2026.06.02 · Source: Nous Research / Decrypt / MarkTechPost
★ Plain-English Highlight

Last issue introduced Hermes Agent's core idea (persistent memory + self-learning skills). This week it shipped its official desktop app (v0.15.2, Public Preview). macOS, Windows, and Linux, one-click install, sharing the same agent core as the CLI. 180K GitHub stars in four months, the fastest-growing open-source agent of 2026.

Practical#06
Anthropic Billing Overhaul + Model Retirement: What Changes on 6/15
📌 Effective 2026.06.15 · Source: Anthropic docs
★ Plain-English Highlight

Starting 6/15, subscription credits split into two pools: interactive usage stays the same; Agent SDK / claude -p / GitHub Actions get a separate credit pool (Pro: $20/mo at API rates). Same day, original Sonnet 4.0 and Opus 4.0 (May 2025 vintage) retire. Opus 4.6 / 4.7 / 4.8 are completely unaffected.

💡 So What

If you only use the web app or terminal chat, you will notice nothing. Action required only for API projects still calling the original 4.0 model IDs (migrate before 6/15) and Agent SDK automation users (note the new credit pool limits).

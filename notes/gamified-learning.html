一套提示词，让 AI 把任何学科变成互动游戏

给孩子复习也好，自己学新东西也好。把一个你陌生的知识，用一个你熟悉的场景重新讲一遍，十分钟出一个可以玩的 HTML 页面。

2026.06 · 实操 · 可直接复制使用
Idea
核心思路：用熟悉的语境解释陌生的知识

孩子觉得分数运算无聊，但每天帮你量米做饭。AI 能做的事情是：用做饭的场景重新讲一遍分数，然后把它变成一个有进度条、有即时反馈、有通关奖励的互动页面。

这个方法的关键不是「让 AI 出题」，而是让 AI 用学习者已经熟悉的生活经验做类比。同样的知识点，换一套语境讲，理解速度完全不同。

下面这套提示词，一次复制给 Claude 或 ChatGPT，它会直接输出一个单文件 HTML 页面，浏览器打开就能用，不需要任何技术背景。

Prompt
直接复制的提示词模板
复制以下内容，修改【】里的部分

我想用「玩着学」的方式掌握【主题，例如：分数运算 / 美国历史 / 基础理财】。

学习者的年龄/水平：【例如：小学四年级 / 初中生 / 零基础成人】

我的背景/熟悉的语境是【例如：做饭 / 打篮球 / 网上购物 / 开车】，请尽量用这个语境打比方来讲解。

目标深度：【能上手实操 / 建立概念 / 应对考试】

请帮我做一个单文件 HTML 网页，一套互动讲解，要求：

1. 进阶路径：把这个主题拆成 3-6 个有先后依赖的关卡，先易后难，像一棵技能树。

2. 嚼碎：每个关卡先用我熟悉的语境做一段三句以内的极简讲解，再配 2-3 道不同形式的小题（单选 / 配对 / 排序 / 真假速判），每题答完立刻给对错 + 一句为什么。

3. 应用：最后给一个「练习场」，让我用刚学的知识做一个贴近真实的决定，并给我反馈。

4. 体验：有进度条、即时反馈、过关激励，视觉风格统一。

5. 难度刚好踩在「有点费劲但做得出来」之间，别太简单也别劝退。

先给我整体关卡大纲让我确认，再逐关生成。

用 Claude Code 或者 Claude Chat 的 Artifacts 功能效果最好，它会直接输出可运行的 HTML。ChatGPT 也能用，让它输出完整代码后存成 .html 文件，双击打开即可。

Examples
给你几个现成的搭配灵感

【主题】和【语境】的搭配是这套方法的灵魂。下面是几个跟我们群友生活相关的例子，直接替换到提示词的【】里就能用。

分数与比例运算 小学 3-5 年级
语境：做饭量米、调味料配比
半杯米加水的比例是多少？食谱说 4 人份，今天只做 2 人份，盐减多少？孩子天天看你做饭，这些数字对他们来说不是抽象的。
美国政府三权分立 初中 Civics
语境：学校里的角色分工
校长定规则像国会立法，老师执行规则像行政分支，学生会申诉像司法审查。用孩子每天待的地方来解释他们课本上的概念。
基础理财：预算与储蓄 自己学 / 亲子
语境：做饭 / 超市购物
每周买菜预算 $150，怎么分配蛋白质、蔬菜、零食的比例？哪些是必须开支哪些可以砍？理财的底层逻辑跟做一周菜单是一回事。
英语写作结构 初高中 ELA
语境：搭乐高 / 盖房子
Topic sentence 是地基，supporting details 是墙壁，conclusion 是屋顶。少了哪一块房子都站不住。段落结构用建筑的逻辑讲，比背模板有用。
职场数据分析入门 自己学
语境：看孩子的成绩单
平均分、中位数、趋势、离群值——这些统计概念你在看孩子成绩单的时候其实一直在用，只是不知道它们的名字。用成绩单数据学 Excel 数据分析，零基础也能上手。
中文阅读理解 中文学校 / 亲子
语境：看动画片 / 讲故事
用孩子看过的中文动画情节做阅读理解题。谁是主角？他为什么做这个选择？如果换一种做法会怎样？阅读理解不是课本专利。
How To
具体怎么操作
1
复制提示词，填入你的主题和语境
想清楚两件事：孩子（或自己）要学的是什么，他/她日常最熟悉的场景是什么。两者越具体越好。「做饭」比「生活」好，「周末篮球训练」比「运动」好。
2
发给 AI，先看大纲
提示词最后一句要求 AI 先给大纲再生成内容。看看关卡拆分是否合理，难度是否合适，语境类比是否贴切。不满意就让它调整。
3
确认后让 AI 生成完整 HTML
Claude 的 Artifacts 会直接在右侧面板渲染出来，可以马上试玩。如果用其他工具，把输出的代码存成 .html 文件，浏览器双击打开就行。手机也能用。
4
跟孩子一起玩，或者自己闯关
有进度条、有即时反馈、有通关激励。一个主题大概 10-15 分钟能做完。做完一个，换个主题再来一个。
这个方法的底层逻辑是：学习最大的障碍不是内容难，是「跟我无关」。当 AI 用你已经理解的经验去解释你还不懂的知识，理解成本直接降一半。剩下的一半靠练习补，而互动小题提供了练习的场景和即时反馈。

如果做出来效果不错，可以发到群里分享。其他人换一个主题和语境，同样的提示词又能生成一个全新的学习页面。一套模板，无限复用。

源于生活，立足应用。

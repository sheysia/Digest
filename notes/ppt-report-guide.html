核心理念 / Core Principle

你给骨架（大纲/要点），AI 填肉（排版/视觉/文案润色）。
You provide the skeleton (outline, key points). AI fills in the muscle (layout, visuals, polish).

不要让 AI 从零生成你的 PPT 或 report——那样出来的东西千篇一律。你先想清楚"我要说什么、给谁看、要达成什么目的"，再让 AI 帮你做得好看、做得快。

Part 1: AI 做 PPT — 三条路线选一条

路线 A：最快最美 — Gamma（推荐大多数人）
1
输入大纲或文档
在 Gamma 里输入你的要点/大纲，或者直接上传 Word/PDF/粘贴 URL。它会在 60 秒内自动生成完整 deck，包括文字、图片、配色。
🔧 Gamma 免费版有 400 credits · gamma.app
2
用 Agent 对话式修改
Gamma 3.0 的 Agent 功能：用自然语言说"把第三页改成对比表格""整体风格换成深色"，它直接改。不用手动拖。
3
导出 PPTX / PDF / 分享链接
导出 PPTX 可以在 PowerPoint 里继续编辑。注意：Gamma 是卡片式布局，导出到 PPT 后格式可能偏移——建议重要演示导出后在 PPT 里做最终调整。或者直接用 Gamma 链接分享（效果最好）。
路线 B：已有 Canva 习惯 — Canva AI
1
Magic Design 一键生成
在 Canva 里选 Presentation → Magic Design → 输入主题，自动选模板、配图、排版。如果已经有 Canva Pro（很多人通过公司或学校有），直接用最省事。
2
手动微调 + 品牌套件
Canva 的优势是模板多、素材多、团队品牌套件好用。适合已经有 Canva 工作流的人——不需要换工具。
路线 C：企业级 + 品牌一致性 — Claude Design / Microsoft Copilot
1
Claude Design — 读你的 design system 出原型
Anthropic 新出的产品。它能读你公司的设计规范，生成保持品牌一致性的原型和 deck。做完可以 handoff 给开发者或导出 PDF/PPTX。适合需要给客户、投资人看的正式场合。
🔧 Claude Pro/Max/Team 用户可用（Research Preview）
2
Microsoft 365 Copilot — 在 PowerPoint 里直接用 AI
如果公司已经部署了 Copilot，直接在 PowerPoint 里用最省事——从 Word 文档生成 PPT、自然语言改 slide、不需要导出/导入。零学习成本。
🔧 需要 Microsoft 365 Copilot 授权（$30/月/人）
💡 选哪条路线？
快速内部分享 → Gamma（链接分享，免费，60 秒出稿）
已有 Canva 习惯 → Canva AI（不换工具，模板多）
正式对外演示 → Claude Design 或 Copilot in PPT（品牌一致性）
纯技术演讲 → 考虑用 Marp（Markdown → Slides），代码控制一切

Part 2: AI 做高级感 Report — 不只是"好看"

"高级感"不是花哨的排版，而是结构清晰 + 视觉节制 + 专业配色。以下是三种产出路径：

路径 A：Claude Artifacts — 你正在看的这份 PDF 就是这么做的
1
给 Claude 你的大纲 + 风格要求
比如："帮我做一份 AI 工具对比报告，A4 排版，蓝绿配色，分四个部分，每部分有卡片式布局。" Claude 直接生成 HTML，排版精确到 pt 级。
2
导出 PDF
Claude 可以直接生成 PDF 文件。你下载就能用。优势：100% 可控——颜色、字体、间距都可以精确指定。
3
适合：内部报告、社群 digest、one-pager、proposal 附件
不需要学任何设计工具。你描述清楚想要什么，Claude 做排版。这份指南本身就是用这个方法生成的。
路径 B：Notion AI — 团队协作文档
1
在 Notion 里写大纲 → AI 扩写
Notion AI 可以帮你扩写段落、改语气（formal/casual）、翻译、总结长文。写完直接在 Notion 里分享，不需要导出。
2
适合：项目文档、会议纪要、团队知识库
免费版支持 10 个协作者，实时同步。如果你的团队已经用 Notion，不需要换工具。
路径 C：Gamma 的 Document 模式
1
输入内容 → 自动生成卡片式长文档
Gamma 不只能做 PPT，也能做长文档——卡片式布局，可以嵌入图表、视频、表格。视觉效果比 Word/Google Docs 好很多。通过链接分享，接收方体验好。
2
适合：项目 brief、客户 proposal、内部 one-pager
不需要排版能力。输入 → 出稿 → 分享链接。但不适合需要精确 PDF 排版或 Word 格式交付的场合。

Part 3: AI 配图 — 流程图、信息图、架构图

Report 和 PPT 里最花时间的往往不是文字，而是图。以前要开 Visio / Lucidchart / draw.io 拖半小时，现在有三个更快的选择：

🍌 Nano Banana Pro（Google Gemini 图像模型）
免费 零门槛
做什么：用自然语言描述，直接生成流程图、信息图、架构图、对比表。文字清晰可读，逻辑自动排版。生成后可以对话式修改（"把箭头改成红色""加一个分支"）。
怎么用：打开 Gemini app 或 Google AI Studio → 输入描述 → 生成 → 迭代修改 → 下载 PNG。
最适合：快速出"能看"的配图，不需要精确到像素。英文文字渲染效果最好——在 DMV 工作场景下正好是优势。中文文字可能出现乱码，建议用英文标注。
🧜 Claude Artifacts（Mermaid / SVG / HTML）
Pro 可用 低门槛
做什么：让 Claude 用 Mermaid 语法或 SVG 生成流程图、架构图、时间线。优势：100% 可编辑——因为底层是代码，改文字/颜色/布局都能精确控制。
怎么用：在 Claude 里说"帮我画一个 data pipeline 架构图，用 Mermaid 格式" → 它直接出图 → 你可以继续修改。
最适合：技术架构图、数据流程图、需要反复修改的场景。中英文都支持。
📊 Canva AI + Beautiful.ai
Freemium 零门槛
做什么：Canva 的 AI infographic generator 从文字生成信息图；Beautiful.ai 的 SmartSlide 自动排版数据图表。两个都是拖拽式，模板多。
最适合：非技术成员需要"设计感强"的配图，给客户/stakeholders 看的场合。
💡 选哪个配图工具？
快速出草图 → Nano Banana Pro（打字就出图，免费）
精确可编辑 → Claude Mermaid/SVG（代码级控制）
设计感最强 → Canva / Beautiful.ai（模板 + 拖拽）

万能 Prompt 模板 — PPT 和 Report 通用

角色：你是一个专业的 [行业] presentation designer。
上下文：这是给 [受众] 看的 [文档类型]，目的是 [达成什么]。
任务：基于以下大纲生成 [PPT/report/one-pager]：
　[你的大纲/要点]
格式：[卡片式/slide 式/A4 长文档]，[配色偏好]，[中英双语/纯英文]。
约束：每 slide 不超过 [X] 个要点。避免大段文字。图表优先于表格。

⚠️ 常见坑
Gamma 导出 PPT 格式偏移 → 重要演示一定在 PowerPoint 里做最终检查和微调。

Nano Banana 中文乱码 → 配图里的文字尽量用英文。需要中文标注的图建议用 Claude SVG 或 Canva。

AI 生成的 report 千篇一律 → 一定先写大纲再让 AI 填充，不要"帮我写一份关于 X 的报告"——要给具体的结构和论点。
